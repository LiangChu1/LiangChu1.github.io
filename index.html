Projects

Project 1: Amazon Clone

In this project, I made a full-stack clone of Amazon's E-commerce website. I was able to make the front-end part of the website using JavaScript and CSS as well as make the back-end part of the website using JavaScript, Stripe's SQL database and Firebase's NoSQL database. For the front-end side of the website, I made the home, login, checkout, payment, and orders page. Futhermore, I made it to where you can add and remove products to the shopping basket. For the back-end side the the website, I made it to where you can view and keep track of a certain user's order history using Firebase' NoSQL database. Moreover, I made the website keep track of everyone's transactions using Stripe's SQL database.

Project 2: Conway's Game of Life Simulation

In this project, I created a real-time graphic rendering of Robert Conway's Game of Life in Processing. I was able to achieve this using Processing's unique drawing functionality, mouse and keyboard recognition, and specified algorithms that Conway used to create the game. In addition, I added some features to the program to make it more user friendly. For example, I included a way that will show the user a random simulation of the game. I also created a way to insert certain patterns, that are synonymous with the game due to how they act, to display on the board. Including the feature to insert certain patterns, I added another feature in which the user can rotate them before placing them on the grid. Furthermore, I implemented a way that would show the next frame/generation of the pattern when a certain key being pressed/held down. Lastly, I incorporated the features that would then pause, unpause, and clear the board.

Project 3: GUI Minesweeper Game

In this project, I collaborated in a team of 3 to create fully functional GUI Minesweeper game in Java/JavaFx. As part of this project, I contributed in creating the backtracking algorithm that would be used to find all of the correct spaces on the board that are not mines. The user would be able to see the backtracking algorithm play out by clicking on the solve button that is viewed in the GUI. I also installed a set of functions that would allow the user to select a space on the board and to tell them whether or not they have selected a mine or not. Futhermore, I implemented another function that would give a visual hint to the next correct space on the board. The user would be able to obtain this visual hint by clicking on a hint button that would be viewed in the GUI.
